# Резервное копирование БД

Backup базы PostgreSQL `images_db`. Сами файлы картинок лежат в Docker-томе. Запускать в bash.

## Создать бэкап

bash scripts/backup.sh
Дамп сохраняется в папку `backups/` с датой и временем в имени, например `backup_2026-06-20_190251.sql`.

## Восстановить из бэкапа

bash scripts/restore.sh backups/backup_2026-06-20_190251.sql
Скрипт принимает первым параметром путь до файла с бэкапом.
